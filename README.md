# snapy-cache

Plugin of [snapy](https://github.com/snapyjs/snapy).

Handles the cache file.

## snapy.config

```js
// ./test/snapy.config.js
module.exports = {

  // …

  // default storage of test cache
  cache: "./test/cache.json", // String

  // …

}
```
